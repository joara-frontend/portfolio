## 🛠 Tech Stack
- **Core:** `React`, `TypeScript`, `Vite`
- **Styling:** `Tailwind CSS`
- **Deployment:** `Vercel`
- **Tools:** `Figma` (UI/UX Design & Assets)

<br />

## ✨ Key Features
- **Semantic Markup:** 검색 엔진 최적화(SEO)를 고려한 시맨틱 태그 활용 및 접근성 준수.
- **Interactive UI:** 사용자 인터랙션을 강조한 아코디언 컴포넌트 및 부드러운 트랜지션 적용.
- **Performance Optimization:** 차세대 이미지 포맷과 메타데이터 최적화로 페이지 로딩 속도 향상.
- **Responsive Design:** Mobile-First 전략을 기반으로 다양한 기기 환경에 최적화된 레이아웃 구현.

<br />

## 📂 Project Structure
```text
src/
